# Random-Task-Generator

## Автор: Костантин Сиротин
## Описание:
Приложение для генерации случайных задач по категориям (учёба, спорт, работа) с сохранением истории в JSON и возможностью добавления собственных задач.

## Возможности:
- Генерация случайной задачи.
- Фильтрация по типу.
- История задач.
- Добавление новых задач с проверкой.
- Сохранение/загрузка истории.

## Примеры использования:
1. Запустить main.py.
2. Нажать «Сгенерировать задачу».
3. Добавить свою задачу через форму.
4. Фильтровать задачи по категориям.
5. История сохраняется при закрытии приложения.
